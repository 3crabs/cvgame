# cvgame

Игра арканойд с управлением головой, а не ж...

## Для тех кто умеет в docker
```
docker build -t cvgame .
docker run cvgame
```

## Для тех кто не обломился и установил питон
```
python -m venv venv
source venv/bin/active
pip install -r requirements.txt
python main.py
```
