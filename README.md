# cvgame

Игра арканойд с управлением головой, а не ж...

## Для тех кто умеет в docker
```
docker build -t cvgame .
docker run cvgame
```

## Для тех кто не обломился и установил python3
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```
